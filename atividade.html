HTML Linguagem de Marcação de Hipertexto

Tutorial Para Iniciantes

HTML(Linguagem de Marcação de Hipertexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento(JavaScript).

Agora que você sabe o que é HTML conheça algumas TAGS abaixo:

Tags Utilidades
<html> Define o início e o fim do documento HTML.
<head> Contém metadados sobre o documento.
<title> Define o título da aba do navegador.
<body> Contém o conteúdo da página.
<h1> a <h6> Define cabeçalhos.
<p> Define parágrafos.
<a> Cria um link.
<img> Insere imagem.
<br> Faz quebra de linha.
